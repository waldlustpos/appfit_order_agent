# 외부 연동 암호화 모듈 가이드

본 문서는 에이전트, POS 등 외부 시스템과 안전하게 데이터를 교환하는 방법을 안내합니다. 암호화에는 **AES-256-GCM** 방식이, 데이터 해싱에는 **SHA-512** 방식이 사용됩니다.

# 사용 목적

- 암호화 과정
  - 에이전트 WebSocket 서버 연결: Authorization 헤더에 project API key를 암호화하여 전송
- 복호화 과정
  - Waldlust Platform Core API: project API key 검증 API (/v0/auth/api-key/validate) 응답의 API key 필드를 암호화하여 제공
- ※ 프로젝트가 고도화 됨에 따라 추가로 사용하는 포인트가 늘어날 수 있습니다.

## 핵심 요구사항

- **공유 비밀키 (`aesKey`)**: 서버와 클라이언트는 반드시 동일한 `aesKey` 값을 공유해야 합니다. 이 키는 **32바이트** 길이의 AES 키를 생성하는 데 사용되며, 해당 값은 작업 진행 전 따로 전달드리겠습니다.

---

## 1. 암호화 및 복호화 (AES-256-GCM)

데이터 암호화는 `AES/GCM/NoPadding` 명세를 따릅니다.

### 암호화된 데이터 구조

암호화된 최종 결과물은 다음 구성 요소들을 Base64로 인코딩한 문자열입니다.

`Base64( IV + EncryptedData + AuthTag )`

- **IV (Initialization Vector)**: 12바이트 길이의 고유한 값입니다. 매 암호화마다 새롭게 생성되어야 합니다.
- **EncryptedData**: 원본 데이터를 암호화한 결과입니다.
- **AuthTag (Authentication Tag)**: 16바이트(128비트) 길이의 인증 태그입니다. 데이터가 위변조되지 않았음을 검증하는 데 사용됩니다.

### 언어별 예제 코드

#### ✅ **Python**

`cryptography` 라이브러리가 필요합니다. (`pip install cryptography`)

```python
import os
import base64
from cryptography.hazmat.primitives.ciphers.aead import AESGCM

# 서버와 공유된 32바이트 AES 키
AES_KEY = "your-super-secret-aes-key-string".encode('utf-8')[:32]

def encrypt(plain_text: str) -> str:
    aesgcm = AESGCM(AES_KEY)
    iv = os.urandom(12)  # 12바이트 IV 생성
    # 암호화 (Ciphertext + AuthTag)
    encrypted_bytes = aesgcm.encrypt(iv, plain_text.encode('utf-8'), None)
    # IV와 암호화된 데이터를 결합
    combined = iv + encrypted_bytes
    return base64.b64encode(combined).decode('utf-8')

def decrypt(encrypted_text: str) -> str:
    aesgcm = AESGCM(AES_KEY)
    combined = base64.b64decode(encrypted_text)
    # IV와 암호화된 데이터 분리
    iv = combined[:12]
    encrypted_bytes = combined[12:]
    # 복호화
    decrypted_bytes = aesgcm.decrypt(iv, encrypted_bytes, None)
    return decrypted_bytes.decode('utf-8')

# --- 테스트 ---
plain = "Hello, Waldlust Platform!"
encrypted = encrypt(plain)
decrypted = decrypt(encrypted)

print(f"Original: {plain}")
print(f"Encrypted: {encrypted}")
print(f"Decrypted: {decrypted}")
```

#### ✅ **Node.js**

Node.js 내장 `crypto` 모듈을 사용합니다.

```javascript
const crypto = require('crypto');

// 서버와 공유된 32바이트 AES 키
const AES_KEY = Buffer.from('your-super-secret-aes-key-string').slice(0, 32);
const IV_LENGTH = 12;
const AUTH_TAG_LENGTH = 16;

function encrypt(plainText) {
  const iv = crypto.randomBytes(IV_LENGTH);
  const cipher = crypto.createCipheriv('aes-256-gcm', AES_KEY, iv);
  
  const encrypted = Buffer.concat([cipher.update(plainText, 'utf8'), cipher.final()]);
  const authTag = cipher.getAuthTag();
  
  const combined = Buffer.concat([iv, encrypted, authTag]);
  
  return combined.toString('base64');
}

function decrypt(encryptedText) {
  const combined = Buffer.from(encryptedText, 'base64');
  
  const iv = combined.slice(0, IV_LENGTH);
  const encryptedBytes = combined.slice(IV_LENGTH, combined.length - AUTH_TAG_LENGTH);
  const authTag = combined.slice(combined.length - AUTH_TAG_LENGTH);
  
  const decipher = crypto.createDecipheriv('aes-256-gcm', AES_KEY, iv);
  decipher.setAuthTag(authTag);
  
  const decrypted = Buffer.concat([decipher.update(encryptedBytes, 'binary'), decipher.final()]);
  
  return decrypted.toString('utf8');
}

// --- 테스트 ---
const plain = "Hello, Waldlust Platform!";
const encrypted = encrypt(plain);
const decrypted = decrypt(encrypted);

console.log(`Original: ${plain}`);
console.log(`Encrypted: ${encrypted}`);
console.log(`Decrypted: ${decrypted}`);
```

#### ✅ **Java**

원본 `ExternalCryptoProvider`와 동일한 로직의 Java 버전입니다.

```java
import javax.crypto.Cipher;
import javax.crypto.spec.GCMParameterSpec;
import javax.crypto.spec.SecretKeySpec;
import java.nio.charset.StandardCharsets;
import java.security.SecureRandom;
import java.util.Base64;
import java.util.Arrays;

public class CryptoClient {

    private static final String AES_KEY_STRING = "your-super-secret-aes-key-string";
    private static final SecretKeySpec AES_SECRET_KEY = new SecretKeySpec(
        Arrays.copyOf(AES_KEY_STRING.getBytes(StandardCharsets.UTF_8), 32), "AES"
    );
    private static final String ALGORITHM = "AES/GCM/NoPadding";
    private static final int GCM_IV_LENGTH = 12;
    private static final int GCM_TAG_LENGTH = 128;

    public static String encrypt(String plainText) throws Exception {
        Cipher cipher = Cipher.getInstance(ALGORITHM);
        byte[] iv = new byte[GCM_IV_LENGTH];
        new SecureRandom().nextBytes(iv);

        GCMParameterSpec gcmSpec = new GCMParameterSpec(GCM_TAG_LENGTH, iv);
        cipher.init(Cipher.ENCRYPT_MODE, AES_SECRET_KEY, gcmSpec);

        byte[] encryptedBytes = cipher.doFinal(plainText.getBytes(StandardCharsets.UTF_8));
        
        byte[] combined = new byte[iv.length + encryptedBytes.length];
        System.arraycopy(iv, 0, combined, 0, iv.length);
        System.arraycopy(encryptedBytes, 0, combined, iv.length, encryptedBytes.length);

        return Base64.getEncoder().encodeToString(combined);
    }

    public static String decrypt(String encryptedText) throws Exception {
        byte[] combined = Base64.getDecoder().decode(encryptedText);
        
        byte[] iv = Arrays.copyOfRange(combined, 0, GCM_IV_LENGTH);
        byte[] encryptedBytes = Arrays.copyOfRange(combined, GCM_IV_LENGTH, combined.length);

        Cipher cipher = Cipher.getInstance(ALGORITHM);
        GCMParameterSpec gcmSpec = new GCMParameterSpec(GCM_TAG_LENGTH, iv);
        cipher.init(Cipher.DECRYPT_MODE, AES_SECRET_KEY, gcmSpec);

        byte[] decryptedBytes = cipher.doFinal(encryptedBytes);
        return new String(decryptedBytes, StandardCharsets.UTF_8);
    }

    public static void main(String[] args) throws Exception {
        String plain = "Hello, Waldlust Platform!";
        String encrypted = encrypt(plain);
        String decrypted = decrypt(plain);

        System.out.println("Original: " + plain);
        System.out.println("Encrypted: " + encrypted);
        System.out.println("Decrypted: " + decrypted);
    }
}
```
