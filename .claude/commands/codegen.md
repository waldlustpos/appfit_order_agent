Bash 툴로 아래 명령어를 실행한다:

```
dart run build_runner build --delete-conflicting-outputs
```

실행 후:
- 생성/수정된 `.g.dart`, `.freezed.dart`, `strings.g.dart` 파일 목록을 간략히 요약
- 오류가 발생했다면 원인을 분석하고 수정 방법을 제안
- 오류가 없다면 "코드 생성 완료" 한 줄로 끝낸다
