---
description: 릴리즈 APK 빌드 후 Lightsail 서버에 배포
---

이 명령어는 **비가역적인 서버 배포**를 포함한다. 반드시 아래 2단계 확인을 거친다.

**1단계 — 배포 전 상태 확인:**
Bash 툴로 아래를 실행하고 결과를 사용자에게 보여준다:
```
git log --oneline -3
```
현재 브랜치와 최신 커밋 3개를 보여주고, "이 상태로 배포하시겠습니까? (yes 입력 시 진행)" 라고 물어본다.

**2단계 — 사용자가 yes 입력 시에만:**
Bash 툴로 `./deploy_apk.sh` 를 실행한다.

실행 후:
- 업로드된 APK 파일명과 버전 정보를 요약
- 오류 발생 시 원인 분석 후 수정 방법 제안

사용자가 yes 외 다른 입력을 하면 배포를 취소하고 종료한다.
