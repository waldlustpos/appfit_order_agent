---
description: Slang i18n JSON 변경분 확인 후 strings.g.dart 재생성
---

아래 순서로 실행한다:

1. Bash 툴로 변경된 i18n 파일 확인:
   ```
   git diff --name-only lib/i18n/
   ```

2. 변경된 파일이 있다면 ko/en/ja 간 키 누락 여부를 간략히 비교 (Read 툴로 각 파일을 읽어 최상위 키 목록 비교)

3. Bash 툴로 코드 재생성:
   ```
   dart run build_runner build --delete-conflicting-outputs
   ```

4. 생성된 `lib/i18n/strings.g.dart` 변경 라인 수를 요약하고 완료를 알린다.
누락된 번역 키가 있다면 어떤 파일에 어떤 키가 없는지 목록으로 알려준다.
